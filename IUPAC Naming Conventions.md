## How to name molecules:
1. Identify the longest carbon chain containing the highest-order functional group
	1. For example, the longest carbon chain is 
```smiles
CCCC(CC)CCCO
```
1. Number the chain
2. Name substituents
3. Assign a number to each substituent
4. Complete the name
